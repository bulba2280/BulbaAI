# BulbaAI
Bulba AI - простая LSTM модель.

# установка / installation
Скачайте chat.py и bulbaAI.pth (https://drive.google.com/file/d/1MNncjymb5bX4qA6_ecXweWz4nvNGluFa/view?usp=sharing)

так же установите библеотеки: pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install numpy colorama wikipedia-api feedparser googlesearch-python beautifulsoup4 requests
